## 資料匯出格式

### CSV 匯出
所有工作表都可以匯出為 CSV 格式，方便未來轉移到其他系統。

### JSON API
系統提供 JSON API 介面，可供其他系統整合：
```json
{
  "equipmentId": "EQ-2024-001",
  "name": "輪椅",
  "category": "行動輔具",
  "area": "屏中區",
  "manager": "王小明",
  "location": "1樓倉庫A區",
  "status": "可用"
}
```

## 未來轉移建議

### 轉移到 MySQL/PostgreSQL
```sql
CREATE TABLE equipment (
  equipment_id VARCHAR(50) PRIMARY KEY,
  name VARCHAR(100),
  category VARCHAR(50),
  area VARCHAR(20),
  manager VARCHAR(50),
  location VARCHAR(100),
  status VARCHAR(20),
  last_inventory_date DATE,
  last_inventory_staff VARCHAR(50),
  notes TEXT,
  photo_url TEXT,
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);
```

### 轉移步驟
1. 從 Google Sheets 匯出 CSV
2. 使用資料庫匯入工具載入
3. 驗證資料完整性
4. 切換系統連線
