# 資料匯入指南

## 📥 從 AppSheet 匯出資料

### 步驟 1: 匯出屏北區資料

1. 開啟 AppSheet 屏北區 App
2. 進入資料管理頁面
3. 點選「匯出資料」
4. 選擇格式：**CSV (逗號分隔)**
5. 儲存為：`inventory_north.csv`

### 步驟 2: 匯出屏中區資料

1. 開啟 AppSheet 屏中區 App
2. 進入資料管理頁面
3. 點選「匯出資料」
4. 選擇格式：**CSV (逗號分隔)**
5. 儲存為：`inventory_center.csv`

## 📊 建立 Google Sheets 資料庫

### 步驟 1: 建立試算表

1. 開啟 [Google Sheets](https://sheets.google.com)
2. 建立新試算表
3. 命名為：**輔具盤點系統**

### 步驟 2: 建立工作表

建立以下 4 個工作表（Sheet）：

#### 工作表 1: 輔具清單

**第一列標題（直接複製貼上）：**
```
財產編號	庫存編號	輔具品名	輔具項目分類	輔具項目名稱	放置地點	保管人	狀態	目前動態	輔具來源	捐贈者	原始編號	規格型號	入庫日期	最後盤點時間	當次作為	備註	照片網址	折合市價	輔具價位	建立日期	更新日期
```

#### 工作表 2: 外借記錄

**第一列標題：**
```
外借編號	財產編號	輔具品名	借用人/單位	聯絡人	聯絡電話	外借日期	預計歸還日期	實際歸還日期	外借狀態	經辦人員	簽名圖片	外借用途	備註	建立日期

**第一列標題：**
```

## 🔄 匯入資料

### 步驟 1: 匯入屏北區資料

1. 在 Google Sheets 中，選擇「輔具清單」工作表
2. 點選「檔案」→「匯入」
3. 選擇「上傳」→ 選擇 `inventory_north.csv`
4. 匯入設定：
   - **匯入位置**：附加到目前工作表
   - **分隔符號類型**：逗號
   - **將文字轉換為數字和日期**：否（保持原始格式）
5. 點選「匯入資料」

### 步驟 2: 匯入屏中區資料

1. 在「輔具清單」工作表中
2. 點選「檔案」→「匯入」
3. 選擇「上傳」→ 選擇 `inventory_center.csv`
4. 匯入設定：
   - **匯入位置**：**附加到目前工作表**（重要！）
   - **分隔符號類型**：逗號
   - **將文字轉換為數字和日期**：否
5. 點選「匯入資料」

3. 取代為：`輔具中心-屏北區:`（或屏中區）

## 📝 資料格式說明

### 日期格式
- **AppSheet 格式**：`0108/11/05`（民國年）
- **建議保持原格式**，系統會自動處理

### 當次作為格式
- **AppSheet 格式**：`◼需清潔☐需充電`
- **新系統格式**：
  - 需清潔：`需清潔`
  - 需充電：`需充電`
  - 兩者都需要：`需清潔,需充電`

### 目前動態選項
- `展示中`
- `外借中`
- `維護中`

### 狀態選項
- `展示`
- `待用`
- `租借`
- `報廢`

## 🚨 常見問題

### Q1: 匯入後發現資料重複怎麼辦？
**A:** 
1. 使用「資料」→「移除重複項目」
2. 選擇「財產編號」欄位
3. 點選「移除重複項目」

### Q2: 兩區有相同的財產編號怎麼辦？
**A:** 
這不應該發生。如果發生，請檢查：
1. 是否誤匯入兩次同一區資料
2. 財產編號是否真的重複（需要重新編號）

### Q3: 照片網址欄位是空的？
**A:** 
這是正常的。照片會在使用新系統盤點時自動上傳並填入網址。

### Q4: 如何確認資料筆數正確？
**A:** 
1. 屏北區原始筆數：（查看 CSV 檔案行數 - 1）
2. 屏中區原始筆數：（查看 CSV 檔案行數 - 1）
3. Google Sheets 總筆數應該 = 屏北區 + 屏中區
