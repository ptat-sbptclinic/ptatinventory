### 步驟 3: 匯入屏北區資料

1. 選擇「輔具清單」工作表
2. 點選「檔案」→「匯入」
3. 選擇「上傳」→ 上傳 `inventory_north.csv`
4. 匯入設定：
   - **匯入位置**: 附加到目前工作表
   - **分隔符號類型**: 逗號
   - **將文字轉換為數字和日期**: **否**（重要！保持原始格式）
5. 點選「匯入資料」

**⚠️ 重要：匯入後需要調整欄位對應**

屏北區 CSV 的欄位順序是：
```
庫存編號,狀態,輔具項目分類,輔具項目名稱,輔具品名,放置地點,輔具來源,捐贈者,入庫日期,出庫日期,租借人,財產編號,原始編號,規格型號,折合市價,輔具價位,保管人,備註,目前動態,最後盤點時間,借用人/單位,報表觸發時間,序號,當次作為
```

新系統的欄位順序是：
```
財產編號,庫存編號,輔具品名,輔具項目分類,輔具項目名稱,放置地點,保管人,狀態,目前動態,輔具來源,捐贈者,原始編號,規格型號,入庫日期,最後盤點時間,當次作為,備註,照片網址,折合市價,輔具價位,建立日期,更新日期
```

**需要重新排列欄位！請看下方「方法 2」使用 Python 腳本自動處理。**

## 🚨 常見問題

### Q: 為什麼要調整欄位順序？
**A:** 新系統將「財產編號」作為主鍵放在第一欄，方便查詢和管理。

### Q: 照片網址欄位是空的？
**A:** 這是正常的。照片會在使用新系統盤點時自動上傳並填入。

### Q: 建立日期和更新日期要填什麼？
**A:** 可以統一填今天的日期，或使用入庫日期。

### Q: 兩區有重複的財產編號嗎？
**A:** 理論上不應該。如果有，需要檢查是否資料錯誤。
