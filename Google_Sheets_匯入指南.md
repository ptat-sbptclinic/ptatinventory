## 🚀 Google Sheets 建立步驟

### 步驟 1: 建立新試算表（2分鐘）

1. 開啟 [Google Sheets](https://sheets.google.com)
2. 點選「空白」建立新試算表
3. 將試算表命名為：**輔具盤點系統**

---

### 步驟 2: 建立 4 個工作表（5分鐘）

#### 工作表 1: 輔具清單

1. 將預設的「工作表1」重新命名為：**輔具清單**
2. **不需要手動輸入標題**，直接進行步驟 3 匯入資料

#### 工作表 2: 外借記錄

1. 點選左下角「+」新增工作表
2. 命名為：**外借記錄**
3. 在第一列輸入以下標題（複製貼上）：

```
外借編號	財產編號	輔具品名	借用人/單位	聯絡人	聯絡電話	外借日期	預計歸還日期	實際歸還日期	外借狀態	經辦人員	簽名圖片	外借用途	備註	建立日期
```

4. 選取第一列，點選「格式」→「文字」→「粗體」

#### 工作表 3: 人員清單

1. 新增工作表，命名為：**人員清單**
2. 在第一列輸入標題：

```
人員編號	姓名	區域	電話	Email	職稱	權限	狀態
```

3. 在第二列開始輸入人員資料（範例）：

```
STAFF-001	李衍霖	屏北區	0912-345-678	lee1@example.com	輔具評估人員	評估人員	在職
STAFF-002	李詩妤	屏北區	0923-456-789	lee2@example.com	輔具評估人員	評估人員	在職
STAFF-003	張郁婷	屏北區	0934-567-890	chang@example.com	輔具評估人員	評估人員	在職
STAFF-004	李啟銘	屏中區	0945-678-901	lee3@example.com	輔具評估人員	評估人員	在職
```

**請根據實際人員修改以上資料！**

#### 工作表 4: 盤點記錄

1. 新增工作表，命名為：**盤點記錄**
2. 在第一列輸入標題：

```
記錄編號	財產編號	盤點日期時間	盤點人員	盤點方式	原放置地點	新放置地點	原目前動態	新目前動態	當次作為	是否拍照	照片網址	備註
```

3. 這個工作表暫時保持空白（系統會自動記錄）

---

### 步驟 3: 匯入整合資料（3分鐘）

1. 選擇「**輔具清單**」工作表
2. 點選「檔案」→「匯入」
3. 選擇「上傳」頁籤
4. 點選「瀏覽」，選擇 `merged_equipment.csv`
5. 匯入設定：
   - **匯入位置**: 取代試算表
   - **分隔符號類型**: 自動偵測
   - **將文字轉換為數字、日期和公式**: **否**（重要！）
6. 點選「匯入資料」

**等待匯入完成（約 10-30 秒）**

---

### 步驟 4: 檢查資料（5分鐘）

#### 檢查清單

- [ ] **總筆數**: 537 列（1 列標題 + 536 筆資料）
- [ ] **欄位數**: 22 欄（A 到 V）
- [ ] **第一筆資料**: 財產編號 = `1A0448080400001`
- [ ] **屏北區資料**: 放置地點包含「屏北區」
- [ ] **屏中區資料**: 放置地點包含「屏中區」
- [ ] **無空白列**: 捲動檢查中間沒有空白列

#### 快速檢查方法

   - 應該是第 537 列

2. **檢查屏北區資料**:
   - 點選「F欄」（放置地點）

   - 點選篩選圖示，搜尋「屏北區」
   - 應該顯示 285 筆

3. **檢查屏中區資料**:
   - 在篩選器中搜尋「屏中區」
   - 應該顯示 251 筆

4. **檢查財產編號**:
   - 點選「A欄」（財產編號）
   - 檢查是否有重複（應該沒有）

---

### 步驟 5: 美化工作表（選擇性，3分鐘）

#### 凍結標題列
1. 選取第 1 列
2. 點選「檢視」→「凍結」→「1 列」

#### 設定欄寬
1. 選取所有欄位（點選左上角）
2. 雙擊任一欄位分隔線（自動調整寬度）

#### 標題列格式
1. 選取第 1 列
2. 點選「填滿顏色」→ 選擇淺藍色
3. 點選「文字顏色」→ 選擇深色
4. 點選「粗體」

## 🚨 常見問題

### Q1: 匯入後發現資料亂碼？
**A:** 
1. 刪除工作表內容
2. 重新匯入時，確認編碼為 UTF-8
3. 或使用「開啟」而非「匯入」

### Q2: 匯入後欄位順序不對？
**A:** 
使用 `merged_equipment.csv` 檔案應該不會有這個問題。
如果有，請確認使用的是整合後的檔案，而非原始 CSV。

### Q3: 總筆數不是 536 筆？
**A:** 
1. 檢查是否匯入了正確的檔案
2. 檢查是否有空白列
3. 重新執行 `merge_data.py` 整合資料

### Q4: 人員清單要填什麼？
**A:** 
填入所有會使用系統的評估人員資料：
- 姓名：用於登入和記錄
- 區域：屏中區或屏北區
- 狀態：必須是「在職」才能登入
