# 輔具盤點管理系統 - 需求說明

## 📌 系統目標

整合屏中區和屏北區的展示輔具盤點管理，提供：
- 快速盤點功能（掃碼→拍照→完成）
- 外借管理功能（掃碼→填資料→簽名）
- 月報表產出（可分區域）

---

## 🎯 核心功能

### 1. 輔具盤點功能

**流程：**
```
點擊「輔具盤點」→ 開啟相機 → 掃描財產編號條碼 → 自動拍照 → 完成
```

**自動處理：**
- 記錄盤點時間（自動）
- 記錄盤點人員（當前登入者）
- 自動判斷「當次作為」：
  - 輔具品名包含「電動」→ 勾選「需充電」
  - 其他 → 勾選「需清潔」
  - 可同時勾選兩者
- 照片自動上傳到 Google Drive
- 更新「最後盤點時間」

**可選操作：**
- 更新放置地點（如有變動）
- 更新目前動態（展示中/維護中）
- 新增備註

---

### 2. 外借管理功能

**流程：**
```
點擊「外借輔具」→ 開啟相機 → 掃描財產編號條碼 → 填寫借用資訊 → 手寫簽名 → 完成
```

**必填欄位：**
- 借用人/單位
- 聯絡電話
- 預計歸還時間

**選填欄位：**
- 外借用途
- 備註

**自動處理：**
- 記錄外借日期（當天）
- 記錄經辦人員（當前登入者）
- 輔具「目前動態」改為「外借中」
- 簽名圖片上傳到 Google Drive

**歸還功能：**
- 掃描條碼或從外借清單選擇
- 記錄實際歸還日期
- 輔具「目前動態」改回「展示中」

---

### 3. 月報表功能

**盤點月報表：**
- 選擇區域：屏中區 / 屏北區 / 全部
- 選擇月份：2024年1月 ~ 當前月份
- 產出內容：
  - 該月盤點的輔具清單
  - 盤點統計（已盤點/未盤點數量）
  - 需維護輔具清單（需清潔/需充電）
- 格式：PDF（根據 sample.docx 範本）

**外借記錄統計：**
- 選擇區域：屏中區 / 屏北區 / 全部
- 選擇月份：2024年1月 ~ 當前月份
- 產出內容：
  - 該月外借記錄清單
  - 外借統計（外借中/已歸還/逾期）
  - 借用單位統計
- 格式：PDF

---

## 📊 資料結構

### 輔具清單（主要資料表）

**核心欄位（必填）：**
- 財產編號（唯一ID）
- 輔具品名（CNS15390標準）
- 放置地點
- 保管人
- 目前動態（展示中/外借中/維護中）

**參考欄位（選填）：**
- 庫存編號（ICF平台）
- 輔具項目分類/名稱（ICF舊分類）
- 狀態（展示/待用/租借/報廢）
- 輔具來源（購買/捐贈/借展）
- 捐贈者
- 原始編號（縣府財編）
- 規格型號
- 折合市價、輔具價位

**盤點相關：**
- 入庫日期
- 最後盤點時間
- 當次作為（需清潔/需充電）
- 照片網址
- 備註

## 🔧 技術實現

### 平台選擇
- **後端**：Google Apps Script
- **資料庫**：Google Sheets
- **儲存**：Google Drive
- **前端**：PWA（可安裝到手機）
- **跨平台**：iOS + Android

### 關鍵功能
1. **條碼掃描**：使用手機相機 + ZXing 條碼庫
2. **照片上傳**：拍照後壓縮並上傳到 Google Drive
3. **電子簽名**：Canvas 手寫簽名
4. **PDF 產出**：Google Apps Script 產生 PDF

## 🔄 資料整合

### 屏中區 + 屏北區合併
1. 兩區資料結構完全相同
2. 合併到同一個 Google Sheets
3. 透過「放置地點」欄位區分區域
4. 報表產出時可選擇區域篩選

### 資料匯入步驟
1. 從 AppSheet 匯出屏中區和屏北區 CSV
2. 在 Google Sheets 建立「輔具清單」工作表
3. 匯入屏北區資料
4. 匯入屏中區資料（附加到現有資料）
5. 確認資料完整性
