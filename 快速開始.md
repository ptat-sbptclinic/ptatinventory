# 🚀 快速開始指南

## 三步驟快速部署

### 第一步：建立 Google Sheets（5 分鐘）

1. 開啟 https://sheets.google.com
2. 建立新試算表「輔具盤點系統」
3. 建立 4 個工作表，複製以下標題：

**輔具清單**（工作表1）
```
輔具編號	輔具名稱	類別	區域	管理人員	擺放位置	狀態	最後盤點日期	最後盤點人員	備註	照片網址	建立日期	更新日期
```

**外借記錄**（工作表2）
```
外借編號	輔具編號	輔具名稱	外借單位	聯絡人	聯絡電話	外借日期	預計歸還日期	實際歸還日期	外借狀態	經辦人員	簽名圖片	備註	建立日期
```

**人員清單**（工作表3）- **必須先新增人員才能登入！**
```
人員編號	姓名	區域	電話	Email	權限	狀態
STAFF-001	王小明	屏中區	0912-345-678	wang@example.com	管理員	在職
STAFF-002	李小華	屏北區	0923-456-789	lee@example.com	一般使用者	在職
```

**盤點記錄**（工作表4）
```
記錄編號	輔具編號	盤點日期	盤點人員	盤點方式	原擺放位置	新擺放位置	原狀態	新狀態	備註
```

---

### 第二步：部署 Apps Script（5 分鐘）

1. 在 Google Sheets 點選「擴充功能」→「Apps Script」
2. 刪除預設內容
3. 複製本資料夾中的 `Code_v2.gs` 完整內容並貼上
4. 點選「部署」→「新增部署作業」→「網頁應用程式」
5. 設定：
   - 執行身分：**我**
   - 存取權：**任何人**
6. 點選「部署」並授權
7. **複製部署網址**（重要！）

---

### 第三步：設定前端（2 分鐘）

1. 開啟本資料夾中的 `config.js`
2. 將部署網址貼到 `API_URL`：
```javascript
API_URL: 'https://script.google.com/macros/s/你的網址/exec',
```
3. 儲存檔案

---

## 🎯 開始使用

### 方法 A：直接使用 Apps Script Web App

1. 在 Apps Script 編輯器點選「執行」旁的網址圖示
2. 或直接開啟部署網址
3. 選擇人員登入
4. 開始使用！

### 方法 B：部署到 Netlify（推薦）

1. 註冊 https://www.netlify.com（免費）
2. 拖曳 `frontend` 資料夾到 Netlify
3. 等待部署完成
4. 獲得永久網址

## 📱 安裝到手機

### iPhone/iPad
Safari 開啟 → 分享 → 加入主畫面

### Android
Chrome 開啟 → 選單 → 安裝應用程式

## 📊 匯入現有資料

1. 從 AppSheet 匯出 CSV
2. 在 Google Sheets 點選「檔案」→「匯入」
3. 選擇「附加到目前工作表」
4. 完成！

## 🔄 未來轉移

當您準備升級到大型系統時：

1. 在 Google Sheets 點選「檔案」→「下載」→「CSV」
2. 匯入新系統資料庫
3. 完成！資料結構已標準化，轉移無痛

## 💡 使用技巧

1. **快速盤點**：掃描條碼後直接按「確認盤點」
2. **批次匯入**：直接在 Google Sheets 複製貼上資料
3. **權限管理**：在「人員清單」新增/停用人員
4. **資料備份**：定期下載 Google Sheets 備份

## ❓ 常見問題

**Q: 無法登入？**
A: 確認「人員清單」工作表有資料且狀態為「在職」

**Q: 掃描條碼沒反應？**
A: 確認輔具編號已存在於「輔具清單」中

**Q: 照片上傳失敗？**
A: 檢查相機權限和網路連線

**Q: 手機無法安裝？**
A: iOS 用 Safari，Android 用 Chrome
